Nalton Calculator Documentation

Basic usage: type an expression and press enter or press "evaluate"

Example expression: sin ((b + 2) * 3 + 4)

Operators:

  • Addition: 2 + 3
  • Subtraction: 2 - 3
  • Multiplication: 2 * 3
  • Division: 2 / 3
  • Exponentiation: 2 ** 3 or 2 ^ 3

Functions

  • Trigonometry:
    • Sine: sin x
    • Arc sine: asin x
    • Cosine: cos x
    • Arc cosine: acos x
    • Tangent: tan x
    • Arc tangent: atan x
  • Roots:
    • Square root: sqrt x or q x
    • Cube root: cbrt x or c x

Variables/memory

  • Set variable: x = 5
  • Use variable: x * 2
