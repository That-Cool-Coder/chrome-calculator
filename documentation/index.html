Nalton Calculator Documentation

Basic usage:

Type an expression and press enter or click the "evaluate" button (button only available in fullscreen mode)

Example expressions:

  • 1 + 2
  • 2 + 3 * 4
  • (2 + 3) * 4
  • sin 1
  • sin (1 + 2)
  • $myvalue = sin (1 + 2)
  • sqrt $myvalue
  • $a = $b = 5   (sets both a and b to 5)

Whitespace is largely optional and some functions have abbreviations! The following expressions are still valid:

  • 2+3*4
  • sin15
  • $roottwo=q2

Specifics:

Operators:

  • Addition: 2 + 3
  • Subtraction: 2 - 3
  • Multiplication: 2 * 3
  • Division: 2 / 3
  • Exponentiation: 2 ** 3 or 2 ^ 3

Inbuilt functions:

  • Trigonometry (take radians as input):
    • Sine: sin x
    • Arc sine: asin x
    • Cosine: cos x
    • Arc cosine: acos x
    • Tangent: tan x
    • Arc tangent: atan x
  • Roots:
    • Square root: sqrt x or q x
    • Cube root: cbrt x or c x

Variables/memory:

  • Set variable: $x = 5
  • Use variable: $x * 2

Custom functions:

  • Define function: $cmToInches => $cm / 2.54
  • Run function: $inches = &cmToInches
  • Currently functions do not support arguments (they're a WIP), so if you want them to do anything useful you need to make them use the global variables (all variables are global).
