Nalton Calculator Documentation

Basic usage:

Type an expression and press enter or click the "evaluate" button (button only available in fullscreen mode)

Use the up/down arrow keys to view command history.

Example expressions:

  • 1 + 2
  • 2 + 3 * 4
  • (2 + 3) * 4
  • sin 1
  • sin (1 + 2)
  • $myvalue = sin (1 + 2)
  • sqrt $myvalue
  • $a = $b = 5 sets both a and b to 5

You can input numbers in a variety of ways:

  • 10
  • 10.5
  • .5 (equal to 0.5)
  • 5 * e3 (equal to 5 * 1000 = 5000)

Whitespace is largely optional and some functions have abbreviations! The following expressions are still valid:

  • 2+3*4
  • sin15
  • $roottwo=q2

You can evaluate multiple expressions at once by separating them with semicolons:

  • 3 * 3; 4 * 4 evaluates to 16
  • $x = 3; $x^2 evaluates to 9
  • $y = 42; &SomeFuncThatUsesY evaluates to result of SomeFuncThatUsesY

Specifics:

Operators:

  • Addition: 2 + 3
  • Subtraction: 2 - 3
  • Multiplication: 2 * 3
  • Division: 2 / 3
  • Modulo: 2 % 3
  • Exponentiation: 2 ** 3 or 2 ^ 3

Inbuilt constants:

  • Pi: $pi (3.141592653589793238462643...)
  • e (Euler's number): $e (2.718281828...)
  • Phi (the Golden Ratio): $phi (1.61803...)
  • Silver ratio: $silv (2.41421...)

Inbuilt functions:

  • Trigonometry (take radians as input):
    • Sine: sin x
    • Arc sine: asin x
    • Cosine: cos x
    • Arc cosine: acos x
    • Tangent: tan x
    • Arc tangent: atan x
  • Roots:
    • Square root: sqrt x c or q x
    • Cube root: cbrt x c or c x
  • Other:
    • Absolute value: abs x
    • Logarithm base 10: log x
    • Natural logarithm: ln x
    • Round value to whole number: round x
    • Floor value to whole number: floor x
    • Ceiling value to whole number: ceil x

Variables/memory:

  • Set variable: $x = 5
  • Use variable: $x * 2

Custom functions:

  • Define function: $multiplyTwoNumbers => $0 * $1
  • Run function: $result = &multiplyTwoNumbers(5, 5) (result will be 25)
  • Function arguments are written into variables with the names $0, $1 and so on. Functions can also read and modify global variables if they wish.
